# Управление пользователями и группами

## Пользователи

### Изменение или назначения пароля пользователя
```bash
sudo passwd <user_name>
```

### Проверка групп пользователя
```bash
group <user_name>
```

## Группы

### Добавить пользователя в группу
```bash
sudo usermod -aG <group_name> <user_name>
```
